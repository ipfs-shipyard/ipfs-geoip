Intro

Installable via npm install --save ipfs-geoip, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save ipfs-geoip

You can then require() ipfs-geoip as normal:

const ipfsGeoip = require('ipfs-geoip')

In the Browser

ipfs-geoip should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of ipfs-geoip, including index.js and index.min.js, are included in the /dist folder. ipfs-geoip can also be found on unpkg.com under

  • https://unpkg.com/ipfs-geoip/dist/index.min.js
  • https://unpkg.com/ipfs-geoip/dist/index.js
